# 多角色技能表生成工具（JSON → Excel）

本项目用于处理多个角色的 JSON 数据，按照角色 **敏捷值（F7/H7/…）从高到低排序**，并生成技能表格，输出到 Excel。工具支持拖拽方式运行（打包为 exe 后），可快速批量处理多个角色的数据。

---

## ✨ 功能特性

### ✔ 1. 多角色 JSON 自动识别  
- 自动检测多个 JSON 文件（多人物数据）
- 每个角色独立解析技能、属性等信息  
- 自动根据 JSON 中的 `skillIds` 生成技能列表

---

### ✔ 2. 按敏捷值排序（从左到右）  
- 程序自动读取各角色敏捷字段（如 f7、h7）  
- 敏捷值 *最高* 的角色排在最左侧  
- 输出的 Excel 也是按敏捷顺序排列

---

### ✔ 3. 自动生成 Excel 表格  
程序会生成一个结构化的 Excel，包括：

| 项目 | 内容 |
|------|------|
| 角色名称 | 来自 JSON |
| 属性列 | 敏捷 / 力量 / 体质 / 等 |
| 技能列表 | 根据 skillIds 解析 |
| 合并单元格 | 按照你的要求合并大项 |
| 表头自动生成 | 自动添加角色信息标题 |

---

### ✔ 4. 拖拽运行（已支持打包 EXE）  
EXE 模式支持：

- 将 JSON 文件或文件夹拖到 exe 图标上执行  
- 自动处理并输出 Excel  
- 不需要安装 Python 环境  

（打包命令示例）：

```powershell
pyinstaller -F your_script_name.py
